# Webcomponent

Permet d'avoir des composants réutilisables et intégrés.

Basés sur 4 spécifications :
* __custom elements__ : permet de créer ces propres éléments HTML
* __shadow DOM__ : encapsule le comportement (évite que les spécificité d'un composant ne "leak" pas à l'extérieur) _émulé avec Angular_
* __template__ : HTML non interprété par le navigateur (permet d'avoir de l'HTML à disposition pour le JS)
* __html imports__ : permet d'importer de l'HTML
